## Background
Development of a real-time alarm clock using the I2C serial communication protocol to interface with an EEPROM to store non-volatile memory

## Hardware 

* STM32F429-Discovery 
* Mini-USB
* Pushbutton switch (3)
* Breadboard
* Breadboard wires
* 24FC64F EEPROM
* 4.7k resistor (2)
* 100 ohm resistors (2)
* 10 uF capacitor
